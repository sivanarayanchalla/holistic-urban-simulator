# Holistic Urban Simulator

A modular, agent-based urban simulation framework for modeling interconnected city systems.

## Features
- Modular architecture for urban system components
- PostgreSQL/PostGIS backend for spatial data
- Agent-based modeling using Mesa framework
- Real urban data integration (OSM, city portals)
- Interactive visualization dashboard

## Quick Start

1. **Setup Environment:**
```bash
conda env create -f environment.yml
conda activate urban-simulator
